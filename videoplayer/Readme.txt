Приложение, позволяющее отображать видео на странице.

1) INSTALLED_APPS = ('videoplayer',)
2) В шаблон добавить {% VideoPlayerMedia %} {% VideoFile файл конфиг %}. 

Файл может быть строкой или переменной.
Конфиг не обязательный параметр. Если он не задан, то используется дефолтный конфиг из conf.settings.
